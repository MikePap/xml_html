Acesso ai Nodi

Si può avere accesso ai nodi in tre modi
Con il metodo getElementsByTagName()
Attraversando l'albero gerarchiale
Navigando tra i nodi dell'albero usando le relazioni
x.getElementsByTagName("title"); // restituisce tutti gli elementi 'title' dell'elmento 'x'
xmlDoc.getElementsByTagName("title"); // restituisce tutti gli elemnti 'title' del documento
xmlDoc è fa riferimento al documento medesimo (vedere file loadxmldoc.js )
xmlDoc=loadXMLDoc("books.xml"); // carica il file "books.xml"
x=xmlDoc.getElementsByTagName("title"); // tutta la lista di elementi 'title' viene immagazzinata nella variabile 'x'
y=x[2]; // nella variabile 'y' viene conservato il valore del terzo elemento 'title'
