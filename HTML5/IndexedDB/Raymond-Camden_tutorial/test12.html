Update a record

Per aggiornare un record il procedimento è quasi simile a come quando lo si aggiunge. Premettendo di aver definito una proprietà chiamata id si usa il metodo put anzichè il metodo add 

var person = {
	name:name,
	email:email,
	created:new Date(),
	id:someId
}
 
var request = store.put(person);

Delete a record

Per eleminare un record si usa il metodo delete passandogli semplicemente l'identificativo dell'oggetto da eliminare 

var t = db.transaction(["people"], "readwrite");
var request = t.objectStore("people").delete(thisId);
