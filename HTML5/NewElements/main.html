main

L'elemento <main> intende specificare il contenuto principale di un documento. Può essere paragonato al classico <div id='container'> con la differenza che non deve contenere contenuti che si ripetono in altri documenti quali header, side bar, navigation links, footer ecc... Inoltre non può esserci più di un elemento per documento e non può essere discendente di altri elementi quali <articli>, <aside>, <footer>, <header> or <nav>.
E' supportato nei maggiori browser tranne che in Internet explorer 

<main> 
	<h1> Web Browsers </h1> 
	<p> Google Chrome, Firefox, and Internet Explorer are the most used browsers today. </p> 

	<article>
		<h1> Google Chrome </h1>
		<p> Google Chrome is a free, open-source web browser developed by Google, released in 2008. </p>
	<article>

	<article>
		<h1>Internet Explorer </h1>
		<p> Internet Explorer is a free web browser from Microsoft, released in 1995. </p1>
	<article>

	<article>
		<h1> Mozilla Firefox </h1>
		<p> Firefox is a free, open-source web browser from Mozilla, released in 2004. </p1>
	<article>
<main>
