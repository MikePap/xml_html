The section element represents a generic document or application section. A section, in this context, is a thematic grouping of content, typically with a heading.
Examples of sections would be chapters, the various tabbed pages in a tabbed dialog box, or the numbered sections of a thesis.
The section element is not a generic container element. When an element is needed for styling purposes or as a convenience for scripting, authors are encouraged to use the div element instead. A general rule is that the section element is appropriate only if the element's contents would be listed explicitly in the document's outline.

L'elemento section rappresenta un gruppo tematico di contenuti, generalmente con un intestazione (heading).
Esempi di "section" possono essere i capitoli o sezioni numerate di tesi.
Non è uno specifico elemento contenitore. Quando c'è bisogno di un elemento per scopi di styling o di scripting allora è meglio usare i <div> . Una regola generale vuole "section" quando i contenuti degli elementi devono essere elencati esplicitamente.
