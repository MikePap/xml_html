SVG & CSS

Ci sono molte proprietà dei CSS che SVG utilizza per lo styling ed altre che sono proprie degli SVG (vedere qui).

I contenuti grafici di un SVG possono essere collegati ai CSS in tre modi:

  • Mediante un file .css esterno
  • Mediante l'uso dell'elemento 'style'
  • Mediante l'uso dell'attributo style

File .css esterno 


mystyle.css

rect {
	fill: red;
	stroke: blue;
	stroke-width: 3
}

SVG file che fa riferimento a mystyle.css

<?xml version="1.0" standalone="no"?> 
<?xml-stylesheet href="mystyle.css" type="text/css"?> 
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd"> 
<svg xmlns="http://www.w3.org/2000/svg" version="1.1" 
  width="10cm" height="5cm" viewBox="0 0 1000 500" >

	<rect x="200" y="100" width="600" height="300" /> 
</svg>

Elemento <style>

L'elemento <style> è posto all'interno dell'elemento <svg> ma deve far uso del costrutto <![CDATA[ ... ]]> il quale elimina la possilità che si venga a crearre un conflitto con l'uso di alcuni caratteri tipo < 

<?xml version="1.0" standalone="no"?> 
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd"> 
<svg xmlns="http://www.w3.org/2000/svg" version="1.1" 
  width="10cm" height="5cm" viewBox="0 0 1000 500" >

	<defs> 
		<style type="text/css"><![CDATA[
			rect {
				fill: red;
				stroke: blue;
				stroke-width: 3
			}
		]]></style> 
	</defs> 

	<rect x="200" y="100" width="600" height="300" /> 

</svg>

Attributo style 

<?xml version="1.0" standalone="no"?> 
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd"> 
<svg xmlns="http://www.w3.org/2000/svg" version="1.1" 
  width="10cm" height="5cm" viewBox="0 0 1000 500" >

	<rect x="200" y="100" width="600" height="300" 
		style="fill: red; stroke: blue; stroke-width: 3" /> 

</svg>

Incorporati all'interno dell'HTML 

<!DOCTYPE html> 
<html lang="it" > 
	<head> 	
		...
		<style type="text/css" > 
			/* dichiarazione regole */
		</style> 
	</head> 
	<body> 	
		<svg width="10cm" height="5cm" viewBox="0 0 1000 500" >
			/* contenuto SVG */
		</svg> 
	<body> 
</html>

Scala delle priorità

La seguente lista mostra il livello delle priorità dello styling nei SVG. Il livello più basso sovrascrive quello/i precedente, in altre parole ha priorità superiore

  • Stili browser
  • Attributi SVG
  • File esterni
  • Stili del documento
  • Stili inline
  • Animazioni
  • Stili sovrascritti (?)
  • computed styles (? stili calcolati)
