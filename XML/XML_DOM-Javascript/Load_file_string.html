Caricamento di un documento esterno XML e di stringa XML interna al documento

loadxmldoc.js

Per caricare nella pagina un documento XML da un file esterno e poterlo manipolare si utilizza XMLHttpRequest (vedere qui l'oggetto). Invece di scrivere il codice nella pagina volta per volta è preferibile scrivere tale codice in un file .js e richiamarlo mettendolo nel tag <head> così come è stato fatto per tutti i file di questa cartella. Il file .js in questione è loadxmldoc.js e si trova nella cartella Allegati.

loadxmlstring.js

Per poter manipolare una struttura XML creata all'interno di uno script javascript (vedi file Traversing.html) è stato scritto del codice all'interno del file loadxmlstring.js che sarà opportunamente richiamato all'interno del tag <head>.
