SVG

SVG è un linguaggio che permette di realizzare grafica vettoriale bidimensionale in XML
Le immmagini SVG non perdono la loro qualità quando sono zoommate o ridimensionate.
Ogni elemento e attributo possono essere animati.
Possono essere create con qualsiasi text-editor ma ideale sarebbe usare Inkscape
Possono essere stampate con alta qualità a qualsiasi risoluzione
Si integra con altri standard W3C come DOM, XSL, CSS.
Ai tag <svg> creati all'interno del documento HTML possono essere applicati gli stili nei CSS (width, height,border, background) ma non a quelli incorporati

I file SVG possono essere creati con estensione .svg e incorporati in un file HTML grazie ai tag <embed>, <object> e <iframe> oppure inseriti nel file HTML direttamente.
Si può inoltre aprire il file SVG semplicemente linkandolo con il tag <a>. Vedere il file cerchio-rosso.svg dove è spiegato il codice SVG.

Se all'elemento SVG non vengono definite width e height per default gli viene assegnato rispettivamente 300 e 150

Incorporare SVG con <image>

Non risulta essere i metodo migliore perchè non supporta bene le animazioni però è supportato da tutti i browser. Per IE dalla versione 9 

<image src="myGraphic.svg" x="?" y="?" width="?" height="?">

Incorporare SVG direttamente nell'HTML

Inserimento diretto di immagine SVG creata direttamente nel file HTML con il tag <svg> nel quale bisogna inserire il namespace: 
xmlns="http://www.w3.org/2000/svg" version="1.1"
E' supportato In Firefox, Internet Explorer 9, Google Chrome, Opera, and Safari

I valori degli attributi del tag <svg> (width e height) impostati nei CSS sostituiscono (hanno la priorità) su quelli impostati all'interno del tag <svg>
