Shadow

context . shadowColor [ = rgb, esadecimale, ecc.. ]
Definisce il colore dell'ombreggiatura
context . shadowOffsetX [ = float ]
Definisce il valore che l'ombra avrà in larghezza. Se tale valore è negativo l'ombra sarà diretta da destra verso sinistra. Default è 0
context . shadowOffsetY [ = float ]
Definisce la distanza dell'ombra dall'alto. Se tale valore è negativo l'ombra sarà diretta da destra verso l'alto. Default è 0
context . shadowBlur [ = float ]
Definisce un valore di opacità all'ombreggiatura. Default è 0
