The header tag defines an introduction to the document.
A header element is intended to usually contain the section's heading (an h1–h6 element or an hgroup element), but this is not required.
The header element can also be used to wrap a section’s table of contents, a search form, or any relevant logos.

L'elemento header definisce una sezione che di solito introduce il resto dellla pagina.
E' inteso come contenitore della sezione headings, ma ciò non è obbligatorio.
Può essere usato per contenere sezioni di contenuti di tabelle, un form di ricerca o dei loghi importanti.
